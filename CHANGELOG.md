# Changelog

All notable changes to Lumen will be documented in this file.

## [0.2.0] - 2026-03-19

### Added
- Gender assumption guardrails — mentors now use neutral language for the user and anyone they mention until gender is explicitly shared
- VERSION and CHANGELOG files for release tracking

### Changed
- Default to Lumen mentor when no argument given to `/project:begin`
- Added Performance mentor to README, AGENTS.md, and CLAUDE.md

## [0.1.0] - 2026-03-16

### Added
- Initial open-source release
- Seven mentor prompts: Lumen, Contribution, Relationships, Vitality, Performance, Spirit, Prosperity
- Council and Coalition session modes
- Two-document memory system (Arc + Session Notebooks)
- Session management workflow (`/project:begin`, `/project:wrap-up`)
- Transcript tracking with structured naming
- Customization guide for creating new mentors
